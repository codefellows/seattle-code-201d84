# Class 07 Lecture Notes

## Constructors

### What are they?

- A special type of function
  - start with keyword `function`
  - acts like a blueprint/'factory' for objects
  - Instantiate objects for us

### Why?

- keeps our dry
- helps to prevent bugs
- gives us consistency/uniform objects
